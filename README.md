# MX5 黑快

Mods I'll probably regret doing to my Talaria MX5

This repository is set up as a Jekyll blog using the [Tale theme](https://github.com/chesterhow/tale) with automatic light/dark mode support based on system preferences.

## Blog

Visit the blog at: [https://stealinglight.github.io/MX5](https://stealinglight.github.io/MX5)

## Local Development

To run the blog locally:

1. Install Ruby and Bundler
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Open [http://localhost:4000/MX5](http://localhost:4000/MX5) in your browser

## Features

- **Tale Theme**: A minimal Jekyll theme for storytelling
- **Automatic Dark Mode**: Uses CSS media queries to detect system color scheme preference
- **Responsive Design**: Mobile-friendly layout
- **SEO Optimized**: Built-in SEO tags and sitemap
- **RSS Feed**: Automatic feed generation

## Content

The blog includes:
- Modification guides and tutorials
- Product reviews
- Real-world performance testing
- Tips and lessons learned
